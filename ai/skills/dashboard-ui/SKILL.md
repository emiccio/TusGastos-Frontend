# Dashboard UI Skill

This skill generates dashboard interfaces for SaaS financial applications.

Examples:

- main dashboard
- analytics pages
- financial summaries
- transaction overviews

---

# Dashboard Goals

The dashboard must communicate information quickly.

Users should understand their financial state within seconds.

---

# Layout

Typical dashboard layout includes:

Metric cards

Charts

Recent activity lists

Category summaries

All elements should align to a consistent grid.

---

# Metric Cards

Important values should appear as cards.

Examples:

Total Balance

Monthly Expenses

Income

Savings Rate

Cards should include:

- title
- main number
- optional change indicator.

---

# Charts

Charts should visualize spending trends.

Examples:

- monthly spending
- category breakdown
- balance over time

Charts should be simple and readable.

Avoid overly decorative chart styles.

---

# Recent Activity

Include lists such as:

Recent Transactions

Recent Expenses

Each row should include:

- description
- category
- date
- amount.

---

# Visual Hierarchy

Numbers are the most important element.

Balance and totals should be visually emphasized.

Use spacing and card grouping to guide attention.

---

# Performance

Dashboards should avoid unnecessary animations.

Rendering should remain efficient.
