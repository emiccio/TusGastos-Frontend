# Forms UI Skill

This skill generates forms used in SaaS productivity and financial applications.

Examples:

- add expense
- edit expense
- profile settings
- category management
- payment method forms

---

# UX Principles

Forms must be:

- quick to complete
- clearly labeled
- logically grouped

Users should be able to complete common actions in seconds.

---

# Form Layout

Prefer vertical forms with clear grouping.

Group related inputs using sections.

Example for expense form:

Amount

Category

Date

Payment Method

Notes (optional)

---

# Field Types

Use appropriate input types:

- number input for amounts
- date picker for dates
- select dropdown for categories
- textarea for notes.

---

# Validation

Forms should include clear validation messages.

Errors should appear near the input field.

---

# Primary Actions

Forms must have clear action buttons.

Examples:

Save Expense

Update Profile

Create Category

Primary buttons should stand out visually.

---

# Component Usage

Use shadcn components:

- Form
- Input
- Select
- Textarea
- Button
- Label

Avoid overly complex form layouts.

Clarity and speed are the priority.
